# PCCCS495 – Term II Project
## Project Title
Task Management Android Application Development Using Object-Oriented Programming
---

## Problem Statement (max 150 words)
Managing daily tasks efficiently is a common challenge for individuals due to lack of proper organization tools. Traditional methods such as writing tasks on paper are inefficient and prone to loss or errors. This project aims to develop a simple and efficient task management application that allows users to digitally store, update, and manage their tasks. The application focuses on usability and structured design using Object-Oriented Programming principles.

## OOP Concepts Used

- Abstraction: Implemented using interface for task operations
- Inheritance: AdvancedTaskManager extends TaskManager
- Polymorphism: Method overriding used in task operations
- Exception Handling: Handles invalid inputs and operations
- Collections / Threads: ArrayList used to store tasks

---

## Proposed Architecture Description
The application follows a modular architecture. The Task class represents the data model containing task details. The TaskManager class handles all business logic such as adding, updating, and deleting tasks. The AdvancedTaskManager extends functionality using inheritance. The system uses Java Collections (ArrayList) for storing task data dynamically.
